#  Chat en Tiempo Real

Trabajo práctico: Sistema de chat colaborativo con WebSocket.

## Descripción
Aplicación web (SPA) que permite a múltiples usuarios comunicarse en tiempo real mediante WebSockets.  
Al conectarse, se asigna automáticamente un nombre de usuario temporal (ej. Usuario_123).  
Los mensajes se comparten instantáneamente entre todos los clientes conectados.

## Requisitos
- Node.js (versión 12 o superior)
- Navegador web moderno
